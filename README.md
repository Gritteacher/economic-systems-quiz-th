# คลังคำถามระบบเศรษฐกิจ

เว็บ static สำหรับสุ่มคำถามภาษาไทยเกี่ยวกับระบบเศรษฐกิจและอุดมการณ์ 4 หมวด หมวดละ 30 ข้อ รวม 120 ข้อ

- ทุนนิยม
- สังคมนิยมประชาธิปไตย
- คอมมิวนิสต์
- ระบบผสม

## การเปิดใช้งาน

เปิดไฟล์ `index.html` ได้โดยตรง หรือใช้ local server:

```bash
python3 -m http.server 4173
```

แล้วเปิด `http://localhost:4173`

## GitHub Pages

ตั้งค่า Pages ใน GitHub เป็น branch `main` และ folder `/root` เพื่อเผยแพร่เว็บนี้ได้ทันที
