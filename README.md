# ระบบเศรษฐกิจ

เว็บ static สำหรับสุ่มคำถามภาษาไทยเกี่ยวกับระบบเศรษฐกิจ 4 หมวด รวม 135 ข้อ พร้อมเฉลย

พัฒนาโดย ครูไต๋ กฤษณพล ทองอุ่น

- ทุนนิยม 30 ข้อ
- สังคมนิยมประชาธิปไตย 35 ข้อ
- คอมมิวนิสต์ 35 ข้อ
- ระบบเศรษฐกิจแบบผสม 35 ข้อ

หน้า `index.html` เป็นหน้าเลือกหมวด และหน้า `quiz.html` เป็นหน้าแสดงคำถามของหมวดที่เลือก

## การเปิดใช้งาน

เปิดไฟล์ `index.html` ได้โดยตรง หรือใช้ local server:

```bash
python3 -m http.server 4173
```

แล้วเปิด `http://localhost:4173`

## GitHub Pages
